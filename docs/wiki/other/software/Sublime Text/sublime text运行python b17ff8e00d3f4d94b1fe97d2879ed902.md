# sublime text运行python

- 安装完anaconda插件之后，可以直接运行Python脚本。但它可能使用的是python2版本，修改配置文件使其使用Python3：
    - Preferences → Package Settings → Anaconda → Settings - User
        
        ```python
        "python_interpreter": "python3",
        ```
