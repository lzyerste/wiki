# RPM包安装

### **2.1.1 安装**

rpm -i 需要安装的包文件

rpm -iv 需要安装的包文件(显示安装详情)

rpm -ivh 需要安装的包文件(显示安装详情及进度)

### **2.1.2 升级**

rpm -U 需要升级的包文件

rpm -Uvh 需要升级的包文件(显示升级详情及进度)

### **2.1.3 卸载**

rpm -e 需要卸载的软件包

注：如果其他程序依赖要卸载的包,系统会提示不能删除,若需要强行删除加上--nodeps,

会强制删除,但可能导致依赖于它的软件不能运行。

### **2.1.4 查看已安装**

rpm -qa

### **2.1.5 查看指定包**

rpm -qa | grep "软件或者包的名字"
